British Rockets

by TheWire

v1.0.0  KSP 1.0.4

***************************************************************************************************************************

Installation:

Requires: 

Klockheed Martian Gimbal to allow single axis gimbals on Gamma first stage engines.
download - https://ksp.sarbian.com/jenkins/job/km_Gimbal/lastSuccessfulBuild/artifact/

Delete all previous installs of British Rockets, both GameData folder and Ships/VAB .craft files.

Unzip GameData/British Rockets to [KSP_directory]/GameData/

Unzip Ships/VAB .craft files to [KSP_directory]/Ships/VAB or [KSP_directory]/saves/[save]/Ships/VAB
